# Analytical Approximations for Project Risk Contingency Reserves

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Python 3.8+](https://img.shields.io/badge/python-3.8+-blue.svg)](https://www.python.org/downloads/)

This repository contains the code and data accompanying the paper:

> **Beyond PERT and Monte Carlo: Analytical Approximations for Project Risk Contingency Reserves with Fat-Tailed Distributions**

## Overview

Traditional project risk quantification relies on Monte Carlo simulation, which can be computationally intensive and obscures the mathematical structure of the problem. This work demonstrates that **closed-form approximations** based on the Cornish-Fisher expansion can replace simulation in most practical cases, achieving accuracy within 0.5% for typical risk registers.

### Key Contributions

1. **Critique of PERT**: We argue against the PERT distribution's bounded support, which systematically underestimates tail risks
2. **Log-normal alternative**: Risk impacts modeled with log-normal distributions, parameterized via P10/P90 expert estimates
3. **Closed-form quantiles**: Derivation of analytical expressions for contingency reserves using moment-matching and Cornish-Fisher expansion
4. **Validation**: Numerical experiments showing <2% error vs Monte Carlo across diverse scenarios

## Usage

### Quick Start

```python
from src.risk_analysis import Risk, compute_aggregate_moments, cornish_fisher_quantile

# Define a risk
risk = Risk(
    id="T01",
    name="Scope creep",
    probability=0.40,
    q10=15000,      # 10th percentile of impact
    q90=45000,      # 90th percentile of impact
    is_threat=True  # True for threats, False for opportunities
)

# Create a list of risks
risks = [risk, ...]  # Add more risks

# Compute analytical moments
moments = compute_aggregate_moments(risks)
print(f"Mean: {moments['mean']:,.0f}")
print(f"Std Dev: {moments['std']:,.0f}")
print(f"Skewness: {moments['gamma1']:.3f}")
print(f"Excess Kurtosis: {moments['gamma2']:.3f}")

# Compute P90 contingency reserve
p90 = cornish_fisher_quantile(
    alpha=0.90,
    mean=moments['mean'],
    std=moments['std'],
    gamma1=moments['gamma1'],
    gamma2=moments['gamma2']
)
print(f"P90 Contingency: {p90:,.0f}")
```

### Using Your Own Risk Register

Prepare a CSV with columns:
- `ID`: Risk identifier
- `Name`: Risk description
- `Probability`: Probability of occurrence (0-1)
- `P10`: 10th percentile of impact if risk occurs
- `P90`: 90th percentile of impact if risk occurs
- `Type`: "Threat" or "Opportunity"

Then load and analyze:

```python
import pandas as pd
from src.risk_analysis import Risk, compare_methods

# Load your risk register
df = pd.read_csv('your_risk_register.csv')

# Convert to Risk objects
risks = [
    Risk(
        id=row['ID'],
        name=row['Name'],
        probability=row['Probability'],
        q10=row['P10'],
        q90=row['P90'],
        is_threat=(row['Type'] == 'Threat')
    )
    for _, row in df.iterrows()
]

# Compare methods
results = compare_methods(risks, alpha=0.90)
print(f"Cornish-Fisher P90: {results['cf_quantile']:,.0f}")
print(f"Monte Carlo P90: {results['mc_quantile']:,.0f}")
print(f"Relative Error: {results['relative_error']:.2%}")
```

## Mathematical Background

### Model

Each risk $i$ contributes:
$$X_i = B_i \cdot D_i \cdot S_i$$

Where:
- $B_i \sim \text{Bernoulli}(p_i)$ — risk occurs or not
- $D_i \sim \text{LogNormal}(\mu_i, \sigma_i^2)$ — impact magnitude
- $S_i \in \{-1, +1\}$ — threat (+1) or opportunity (-1)

Total risk: $X = \sum_{i=1}^{n} X_i$

### Log-Normal Parameterization

From expert percentiles P10 and P90:
$$\mu_i = \frac{\ln q_{90} + \ln q_{10}}{2}$$
$$\sigma_i = \frac{\ln q_{90} - \ln q_{10}}{2 \cdot z_{0.90}} \approx \frac{\ln(q_{90}/q_{10})}{2.563}$$

### Cornish-Fisher Expansion

The $\alpha$-quantile is approximated by:
$$q_\alpha \approx \mu + \sigma \cdot z_{CF}$$

Where:
$$z_{CF} = z_\alpha + \frac{z_\alpha^2 - 1}{6}\gamma_1 + \frac{z_\alpha^3 - 3z_\alpha}{24}\gamma_2 - \frac{2z_\alpha^3 - 5z_\alpha}{36}\gamma_1^2$$

### Domain of Validity

The approximation is reliable when:
- $|\gamma_1| < 2.5$ (skewness)
- $\gamma_2 < 6$ (excess kurtosis)

For typical risk registers with $n \geq 10$, these conditions are almost always satisfied.
